Render:addPics(spritelist, x, y, w, h, s, f, xo, yo)
This function adds 'sprites' to the entity using a sprite list or a whole image

Parameters:

  • spritelist Drawable
  • x number the number of cells in a sheet horizontally
  • y number number of vertical sheet cells
  • w number width of one sheet cell
  • h number height of one sheet cell
  • s number starting point of recording in the sprite array
  • f number number of frames entered
  • xo number x offset
  • yo number y offset
