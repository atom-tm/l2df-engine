Class l2df.core.manager.resource

Resource manager

Info:

  • Copyright: Atom-TM 2019
  • Author: Kasai

Functions

Manager:add(resource, temp) Saves the resource to the Manager
Manager:addById(id, resource, temp) Stores the resource in the Manager using a unique identifier
Manager:remove(resource) Removes the specified resource from the Manager
Manager:removeById(id) Removes the specified resource from the Manager by Id
Manager:clearTemp() Removes all resources marked as temporary from the Manager
Manager:getId(resource) Gets the resource Id from the Manager
Manager:get(id) Returns a resource from the Manager by Id
Manager:has(id) Checks the availability of the resource in the Manager by the Id
Manager:load(filepath, reload, id) Adds a supported file type to the Manager by loading it from a path
